## PROJECT CODENAME
**Mnemosyne** — Greek goddess of memory, mother of the nine Muses.
The name is intentional. Memory is not a feature. It is the foundation of intelligence.

---

## ONE-LINE DEFINITION
> Mnemosyne is a local-first adaptive cognitive state engine that gives AI systems persistent, structured, evolving workspace cognition — not memory replay, but genuine continuity of understanding.

---

## WHAT THIS IS NOT
- Not a chat history tool
- Not a vector database wrapper
- Not a RAG system
- Not an LLM fine-tuning pipeline
- Not a note-taking app with AI

## WHAT THIS IS
- A cognitive infrastructure layer
- A workspace-scoped state machine
- A structured knowledge graph with temporal versioning
- A hybrid retrieval orchestrator
- A privacy-first local intelligence engine
